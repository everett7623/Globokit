## 功能边界

- 仅提供外贸与物流常用长度、重量、体积单位换算，不接入汇率、实时 API 或用户数据存储。
- 输入值必须为非负有限数；不同量纲之间不允许换算。
- 换算逻辑保持纯函数，同一输入始终得到同一结果。
