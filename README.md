# Globokit - 外贸实用工具集

一个现代化的外贸工具网站，提供多种实用的文本处理工具。

## 功能特性

- 🔢 **人民币大写转换** - 将数字金额转换为中文大写格式
- 🔤 **英文大小写转换** - 快速转换英文文本的大小写格式
- ⚠️ **特殊字符检查** - 检查并转换文本中的特殊字符
- 🔣 **数字转英文** - 将数字转换为英文表达形式
- 🈶 **中文转拼音** - 将中文文本转换为拼音

## 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS + shadcn/ui
- **语言**: TypeScript
- **部署**: Vercel
